Project Brief

The project consisted programming a NAO robot to play the music of a movie and dance.
This was part of an integration project for a teacher of an African school.

Key Features
  • Playing music
  • Dancing following a choreography
  • Changing eyes color
  • Tracking and kicking a red ball at a particular part of the song
My Role / Tasks

I was involved in the development of the project from start to end, including tasks such as:

  • Developing a python algorithm using NAO libraries
  • Interacting with different sensors
  • Testing in a simulator
