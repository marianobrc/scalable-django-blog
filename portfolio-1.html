Project Brief

The project consisted in building a custom system for a Financial Services Provider Company.

Service Providers can see their upcoming payments in the system and opt for early pay with different options.

The system calculates the discounts using a custom algorithm.

Accountants can manage the available financial options and approve early payment requests.

Key Features:
  • Multiple User Roles Support: Accountants, Service Providers and Admins
  • A Web Platform for Providers
  • A Back-office interface for Accountants and Admins
