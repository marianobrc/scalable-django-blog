Project Brief

The project consisted in an IoT Device plus a web platform used to monitor and analyze vitals while sleeping.

Key Features
  • Monitoring heartbeats and breath and storing sensor data in the cloud
  • Analyze sensor data to determine the sleep stage and quality
  • Integration with other smart devices such as smart lights (Philips HUE)
  • Sunset / Sunshine simulation
